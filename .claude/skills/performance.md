# Performance Report

## Steps

1. Ask for Twitter handle and timeframe (daily/weekly/monthly/all_time, default all_time)
2. Call `trader_performance_report` tool
3. Present: Composite Score, Rank, Return % (50% weight), Sharpe (30%), Max Drawdown (20%), Win Rate, Trade Count

## Scoring: 50% return, 30% consistency (Sharpe), 20% risk management (max drawdown)
